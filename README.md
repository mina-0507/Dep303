# Лабораторная работа №8
## Вариант 5. Создание приложения: "Финансовый трекер расходов".
### Описание:
Приложение для учёта личных расходов с графическим интерфейсом. Позволяет пользователю добавлять информацию о расходах (сумма, описание, категория, дата), просматривать историю расходов в виде таблицы, удалять записи и получать статистику расходов. Все данные сохраняются в базе данных SQLite.

### Инструкции по запуску:
1. В терминале установить PySide6: py -m pip install PySide6 (python3 -m pip install PySide6 - для Linux)
2. Запустить файл main.py: py main.py (python3 main.py - для Linux)

### Краткая справка:
- Добавление расхода: заполните поля слева и нажмите "Добавить расход"
- Удаление расхода: выберите строку в таблице и нажмите "Удалить выбранное"
- Просмотр статистики: отображается автоматически справа вверху
- Кнопка "Обновить" перезагружает данные

### Требования лабораторной работы:
| Требование | Реализация |
|------------|------------|
| GUI приложение | PySide6 |
| ОО парадигма | Классы Database и FinanceTracker |
| Собственные исключения | FinanceError, InvalidAmountError, EmptyDescriptionError, DatabaseError |
| Хранение данных в БД | SQLite (файл finance.db) |
